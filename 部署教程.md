# 宿舍面试系统 - 部署到 Render 教程

## 你需要准备

- [ ] 一个 GitHub 账号（github.com，用邮箱注册，免费）
- [ ] 一个 Render 账号（render.com，用 GitHub 登录，免费）
- [ ] 本地的代码文件夹

---

## 第一步：安装 Git

1. 去 https://git-scm.com/download/win 下载 Windows 版 Git
2. 安装时一路点 Next 就行（全部默认）
3. 安装完后，Win+S 搜索 `Git Bash`，如果能打开就说明装好了

---

## 第二步：上传代码到 GitHub

按下键盘 `Win + R`，输入 `cmd`，回车。然后逐行输入以下命令：

```bash
# 1. 进入项目文件夹
cd "C:\Users\avera\Desktop\新建文件夹 (3)\dorm-interview"

# 2. 初始化 git
git init
git add .
git commit -m "宿舍面试系统"
```

然后去 github.com：
1. 登录后点右上角 `+` → `New repository`
2. Repository name 填 `dorm-interview`
3. 保持 Public（公开，免费）
4. 不要勾选任何复选框
5. 点绿色的 `Create repository` 按钮

页面上会出现几行命令，复制并执行带有 `git remote add` 和 `git push` 的那几行，类似：

```bash
git remote add origin https://github.com/你的用户名/dorm-interview.git
git branch -M main
git push -u origin main
```

完成后刷新 GitHub 页面，能看到你的代码文件就成功了。

---

## 第三步：部署到 Render

1. 打开 render.com，点 `Sign In` → `Sign in with GitHub`
2. 第一次会问你是否授权，点 `Authorize Render`
3. 登录成功后，点右上角 `New` → `Web Service`
4. 点 `Connect account`（连接 GitHub），选择你刚创建的 `dorm-interview` 仓库
5. 填写配置：

| 配置项 | 填什么 |
|--------|--------|
| **Name** | `dorm-interview`（随便填） |
| **Region** | 选 `Singapore`（新加坡，国内访问最快） |
| **Build Command** | `npm install` |
| **Start Command** | `node server.js` |
| **Instance Type** | **选 Free** |

6. 滚动到最下面，点 `Create Web Service`

---

## 第四步：等待部署

- Render 会自动安装依赖、启动服务
- 大约 2~3 分钟后，页面上方出现绿色的 `Live` 字样
- 上面显示的 URL 就是你的网站地址，格式：`https://dorm-interview-xxxx.onrender.com`
- 把这个链接发给三位舍友，所有人都能用浏览器打开了

---

## 第五步：以后怎么更新

改完代码后，在命令行执行：

```bash
git add .
git commit -m "更新"
git push
```

Render 会自动检测到更新并重新部署，等两分钟就好。

---

## 注意事项

- **冷启动**：免费版 Render 如果 15 分钟没人访问会休眠，下次打开大概要等 30 秒。这是免费版的限制
- **内存**：免费版 512MB，四个人用绰绰有余
- 如果两个月都没人用需要重新激活，去 Render 控制台点一下就行

---

## 补充：添加 Turso 免费云数据库（解决数据丢失问题）

Render 免费版没有持久化磁盘，服务休眠或重新部署后数据会丢失。用 Turso（免费 SQLite 云数据库）彻底解决。

### 1. 注册 Turso

1. 打开 https://app.turso.tech
2. 点 **Sign up with GitHub**
3. 登录后点 **+ Create Database**
4. Database name 填 `dorm-interview`
5. 区域选 `Tokyo`（东京）或 `Singapore`（新加坡）——离国内近
6. 点 **Create**

### 2. 获取连接信息

数据库创建好后：
1. 点一下刚创建的数据库
2. 找到 **Connection URL**，复制（格式：`libsql://dorm-interview-xxx.turso.io`）
3. 点 **Generate Token**（生成密钥），复制这个 token

### 3. 在 Render 添加环境变量

1. 打开 render.com → 你的服务
2. 左边找到 **Environment** 
3. 添加两个环境变量：

| Key | Value |
|-----|-------|
| `TURSO_URL` | 刚才复制的 URL（libsql://...） |
| `TURSO_TOKEN` | 刚才复制的 Token |

4. 点 **Save Changes**，Render 会自动重新部署

从此以后**数据彻底持久化**，无论部署多少次、服务休眠多久，数据都不会丢。
